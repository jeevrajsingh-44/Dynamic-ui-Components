# Music Card

A simple and reusable music card component built using **HTML**, **CSS**, and **Vanilla JavaScript**. The component is generated dynamically through JavaScript, making it easy to create multiple cards with different song information.

## Features

* Dynamic card creation using JavaScript
* Reusable `createCard()` function
* Album cover, song title, artist name, and duration
* Play/Pause button with toggle interaction
* Random button color generation on every click
* Responsive Flexbox-based layout
* Multiple cards generated from a single function

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
