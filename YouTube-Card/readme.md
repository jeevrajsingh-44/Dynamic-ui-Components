# Dynamic YouTube Card

A dynamic YouTube-style video card built using HTML, CSS, and Vanilla JavaScript.

This project was created to practice DOM manipulation by generating all card elements dynamically instead of writing them directly in HTML.

## Features

- Dynamic card creation using JavaScript
- Dynamic thumbnail
- Dynamic video title
- Dynamic channel name
- Dynamic views and upload time
- Duration badge on the thumbnail
- Flexbox-based layout

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (DOM Manipulation)

## Project Structure

```
youtube-card/
│── index.html
│── style.css
│── script.js
│── README.md
```

## Learning Outcomes

Through this project I practiced:

- Creating elements using `createElement()`
- Appending elements to the DOM
- Working with nested DOM structures
- Using Flexbox for layouts
- Positioning elements with `position: relative` and `position: absolute`
- Passing dynamic data through function parameters
